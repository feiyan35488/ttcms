# ttcms [![Build Status](https://travis-ci.org/feiyan35488/ttcms.png?branch=master)](https://travis-ci.org/feiyan35488/ttcms)
## JAVA版CMS 
=====

#### 该项目使用nutz-extends 框架开发

目前有功能：
  * 文章管理和发文章
  * 分类管理
  * 标签管理
  * 支持kindEditor和markdown两种编辑器
  * 有js 生成qrcode
  * 可同步发送文章到 oschina中
  * 
这个项目目前没有更新，后面计划会有大的改动
